# vendor asrmicro

## 介绍

该仓库托管翱捷科技开发的样例代码,包含xts和wifi两个demo的配置文件和hal模块。

## 软件目录

代码路径：

```
device/                               --- 硬件单板相关仓库
├── board/lango                      --- 朗国公司相关单板
└── soc/asrmicro                      --- ASR相关SoC代码

vendor/asrmicro/                      --- vendor_asrmicro 仓库路径
├── wifi_demo                         --- wifi样例
└── xts_demo                          --- xts样例
```
